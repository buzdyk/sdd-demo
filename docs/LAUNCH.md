# Launch Plan

## Current State

A professional tech blog authored by Gregory, intended to be exposed publicly. Architecture is intentionally static. The Vue ecosystem is preferred (Hugo is explicitly out — see [[CHOOSE_STATIC_GENERATOR]]). Deployment will be driven from local for now; CI-driven deploy is iceboxed (see [[GITHUB_ACTIONS_DEPLOY]]).

## Launch Stages

### Stage 1 — MVP

Scope: a domain pointing at a single static page.

- [ ] Pick a Vue-friendly static site generator
- [ ] Build the single landing page
- [ ] Pick a host and register the domain
- [ ] Wire up a local-driven deploy
- [ ] Point the domain at the deployed page

Theme system, post authoring pipeline, and a real article are deferred to post-MVP — see backlog.
