# Glossary

## Services

| Term | Definition |
|------|------------|
| Blog | The static, public-facing professional tech blog authored by Gregory |

## Data Entities

| Term | Definition |
|------|------------|
| Post | A single blog article (Markdown source, rendered to HTML at build time) |

## Infrastructure

| Term | Definition |
|------|------------|
| Static Site | Pre-rendered HTML/CSS/JS — no runtime server, no database |
| Vue | Preferred frontend ecosystem; static-site generator choice will favor Vue-native tooling (e.g. VitePress, Nuxt Content, VuePress) |
| Local Deploy | Deploys are kicked off from the author's machine; CI-driven (GitHub Actions) deploy is iceboxed |
