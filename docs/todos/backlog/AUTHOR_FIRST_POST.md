# Author first post

## Problem

A blog with no posts is just scaffolding. The first post sets the tone for everything that follows and validates the end-to-end author → build → deploy loop.

## Approach

- Pick a topic that is genuinely interesting and short enough to actually finish.
- Draft, edit, run through the full publish pipeline.
- Use this as the smoke test for theme, build, and host.
