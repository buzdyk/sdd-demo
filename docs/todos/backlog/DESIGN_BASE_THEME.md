# Design base theme

## Problem

A professional tech blog needs a consistent, readable visual identity before the first post is published. Theme decisions made post-hoc tend to leak into content and become expensive to change later.

## Approach

- Decide on typography (one body, one heading, one mono) with a clear rationale.
- Define a minimal color palette and spacing scale.
- Sketch home, post, and listing layouts.
- Implement once the static generator is chosen.
